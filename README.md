# Portfolio — Chittibala Teja Nandiswar Reddy

This repository contains a responsive, dark-themed single-page portfolio optimized for GitHub Pages.

Features
- Hero, About, Skills, Projects, Resume and Contact sections
- Dark professional color palette (primary: #2563EB)
- Smooth animations and simple typing effect
- Mobile-first responsive layout

How to deploy
1. This is already pushed to the `main` branch of this repository.
2. GitHub Pages for a user site (username.github.io) is served from the `main` branch by default. Visit: https://nandhuteja.github.io

Editing content
- Edit `index.html` to update text, social links and project details.
- Replace the placeholder images in `assets/` with your project screenshots.

Resume
- The "Download Resume" button links to the provided Google Drive URL. If you prefer the resume hosted in the repo, upload `resume.pdf` to the repository root and update the link in `index.html`.

License
